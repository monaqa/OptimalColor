# OptimalColor

良いカラースキームに求められそうな条件を数式で表現し、
数理に最適なカラースキームを計算するコード。

`script/normal.jl` にあるスクリプトを上から順に実行していくことで最適化が行われる。

得られる配色は以下の通り（最適化の結果は変わる可能性がある）。

- 背景色
  - `#141414`
  - `#142b14`
  - `#431414`
  - `#181434`

- 文字色
  - `#ebebeb`
  - `#3997c0`
  - `#e771cc`
  - `#7e88da`
  - `#509546`
  - `#da7575`
  - `#179f8a`
  - `#9a8427`
  - `#e77335`
